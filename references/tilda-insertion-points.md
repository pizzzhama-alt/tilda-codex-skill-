# Куда вставлять код в Tilda

## CSS

Использовать один из вариантов:

- **Настройки страницы -> Дополнительный CSS**: для стилей только одной страницы.
- **Настройки сайта -> Еще -> HTML-код в HEAD**: для общих стилей на всем сайте, если изменение действительно глобальное.
- **Блок T123 "HTML-код"**: можно вставить `<style>...</style>` на конкретной странице рядом с нужным блоком.
- **Zero Block -> Settings -> Add CSS class**: добавлять собственный класс элементу Zero Block, затем стилизовать его через scoped CSS.

Предпочитать CSS на уровне страницы или блока, а не всего сайта.

## JavaScript

Использовать один из вариантов:

- **Блок T123 "HTML-код"**: вставить `<script>...</script>` после целевого блока или внизу страницы.
- **Настройки страницы -> Дополнительный HTML-код перед `</body>`**: для страницы целиком.
- **Настройки сайта -> HTML-код перед `</body>`**: только для общих решений, которые должны работать на нескольких страницах.

Для каталогов, попапов товаров, меню и форм предпочитать вставку перед `</body>`, потому что DOM может дорисовываться после первичной загрузки.

## HTML

Использовать:

- **T123 HTML block**: для кастомных HTML-фрагментов, виджетов, дополнительных контейнеров.
- **Zero Block HTML element**: если нужно встроить HTML внутри Zero Block.

Не вставлять крупную разметку в глобальный HTML сайта, если она нужна только одной странице.

## Объяснение в ответе

Всегда явно писать:

- где открыть настройку;
- вставлять ли код на страницу или на весь сайт;
- должен ли блок T123 стоять после целевого блока;
- что заменить: `#recXXXXXX`, класс Zero Block, текст, ссылки, цвета.
